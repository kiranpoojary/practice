Three Ways to Insert CSS

There are three ways of inserting a style sheet:
  1. External CSS I
  2. Internal CSS
  3. Inline CSS

1.External CSS : in Head section

<link rel="stylesheet" href="mystyle.css">

2.Internal CSS: in Head section

<style> 
        .p{
            color:red;
        }

        .myClass{
            color:green;
        }

        #myId{
            color:yellow;
        }
</style>

3.inline CSS : in Head section

<p style="color:red;">Hello </p>

IMP NOTE: Cascading Order

What style will be used when there is more than one style specified for an HTML element?

All the styles in a page will "cascade" into a new "virtual" style sheet by the following rules, where number one has the highest priority:
  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default



NOTE:

  1. Multiple Style Sheets: If some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.
