CSS LINKS

With CSS, links can be styled in many different ways.

        /* unvisited link */
        a:link {
          color: red;
        }
        
        /* visited link */
        a:visited {
          color: green;
        }
        
        /* mouse over link */
        a:hover {
          color: hotpink;
        }
        
        /* selected link */
        a:active {
          color: blue;
        }

        a:link {
            background-color: yellow;
          }
          
          a:visited {
            background-color: cyan;
          }
          
          a:hover {
            background-color: lightgreen;
          }
          
          a:active {
            background-color: hotpink;
          }
